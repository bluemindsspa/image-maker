Odoo Sale Order Discount Approval App

Odoo Sales Order Discount Approval App allows restricting discounts on the sale order lines.If user gives more discount against configured amount then that sale order goes for approval for discount manager approval, approval manager can get approval via email with order link and approve the order.

Sale Order Discount Configuration

Users can set the DIscount Configurations.

Create sale Order Quotation

Users can create sale quotations and Send them to Admin.

View and Check the Order Details

After creating a sale order mail will send to Admin and Approved By Admin.
